API

PenguinAnalysis.CellMeasure — Type
CellMeasure(V; celltype=nothing)
CellMeasure(; V, celltype=nothing)
CellMeasure(obj)

Lightweight holder for wet measure V and optional celltype tags.

PenguinAnalysis.region_mask — Function
region_mask(cm::CellMeasure, region::Symbol)

Return a boolean mask selecting active cells in region (:all, :cut, :full).

PenguinAnalysis.lp_error — Function
lp_error(u, u_exact, geom; p=2, region=:all, relative=false)

Compute the weighted discrete global L^p error for one field or an aggregated field tree.

PenguinAnalysis.lp_errors — Function
lp_errors(u, u_exact, geom; p=2, region=:all, relative=false)

Compute weighted discrete L^p errors leaf-by-leaf, preserving tuple tree structure.

PenguinAnalysis.lp_error_report — Function
lp_error_report(u, u_exact, geom; p=2, region=:all, relative=false)

Compute weighted discrete error and return an ErrorReport with diagnostics.

PenguinAnalysis.ErrorReport — Type
ErrorReport

Summary container returned by lp_error_report.

PenguinAnalysis.pairwise_orders — Function
pairwise_orders(errors, hs)
pairwise_orders(errors)

Compute pairwise convergence orders from error sequences, with optional mesh sizes hs.

PenguinAnalysis.overall_order — Function
overall_order(errors, hs)

Return the global log-log slope from a least-squares fit of errors against hs.
